# AliceSW 阅读器

AliceSW 阅读器是一个用于阅读小说的现代桌面应用程序，采用 Python 和 PyQt6 开发。它提供了一个流畅、美观、无广告的阅读体验，并专门适配了部分反爬站点的浏览。

## 功能特性

- **流畅美观的UI设计**: 采用现代扁平化设计，左右分栏布局。
- **小说搜索与分类**: 提供多种分类榜单，支持关键词精准搜索。
- **章节列表浏览**: 快速获取小说的全部章节目录。
- **专属阅读模式**:
  - 内置高性能无头浏览器（基于Chromium），直接加载页面，自动过滤广告和其他杂乱元素。
  - 支持 **字体放大/缩小**。
  - 支持 **夜间/护眼模式** 切换。
  - 完美的中文排版（自动首行缩进，合适的行间距）。

## 环境依赖

- Python 3.8+
- Windows / macOS / Linux (带桌面环境)

## 安装方法

1. 克隆或下载此项目代码。
2. 安装依赖包：

```bash
pip install -r requirements.txt
```

## 运行方法

在项目根目录下执行：

```bash
python main.py
```

## 技术选型

- **PyQt6**: 提供高效稳定的图形用户界面。
- **PyQt6-WebEngine**: 内嵌 Chromium 浏览器，用于渲染章节内容，自带突破常规爬虫防护（如浏览器指纹拦截等）的能力。
- **cloudscraper**: 用于处理初步页面的 Cloudflare 等防护。
- **BeautifulSoup & Regex**: 用于 HTML 解析和信息提取。
